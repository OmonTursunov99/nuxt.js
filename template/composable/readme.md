# Composable
Функциональный часть логических компонентов

## Пример
Должен создаваться папка по директории "pages"

- Базовый
```ts
export function useExample() {

  return {

  };
}
```
- Репозиторий
```ts
export function useExampleFetch() {
  
  return {
    
  };
}
```
- Вспомогательные функции
```ts
export function useExampleHelper() {
  
  return {
    
  };
}
```
